1. Survival_Rate_Pred_using_ML_models.ipynb
Implemented various machine learning models to predict survival rates based on input features. Includes preprocessing, model training, and performance evaluation using metrics like accuracy and confusion matrix.

2. sales-price-prediction-using-ml-xgb.ipynb
Advanced price prediction model using XGBoost, optimized for accuracy and speed. Focused on hyperparameter tuning and feature importance visualization.

3. Rice_Disease_classification.ipynb
Image classification project for detecting rice plant diseases using deep learning. Used CNN models with performance measured through precision, recall, and F1-score.

4. Plant_Pathology_2020_FGVC7_Project.ipynb
Kaggle competition-based project for classifying plant leaf diseases. Applied transfer learning techniques and evaluated results using multi-class classification metrics.

5. E-commerce_data_analysis.ipynb
Exploratory Data Analysis (EDA) on e-commerce data to uncover sales trends, customer behavior, and product insights using visualizations and statistical summaries.

6. Digit_Recognizer.ipynb
Handwritten digit recognition using the MNIST dataset. Built a CNN model in Keras to classify digits from 0 to 9 with high accuracy.

7. heart_disease_project.ipynb
Classification project for predicting the presence of heart disease using patient health data. Compared multiple ML models and visualized important features influencing predictions.
